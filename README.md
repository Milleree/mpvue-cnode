## 本地运行

``` bash
 # 切换到项目目录
 cd project-name

 # 安装依赖
 npm i

 # 启动脚本
 npm run dev

 # 然后用微信开发者工具打开本项目根目录即可。
```


## 其它脚本

```bash
# eslint 检查
npm run lint

# eslint 自动修复
npm run fix

# 生产环境 build
npm run build
```
